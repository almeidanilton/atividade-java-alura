# 💻 Atividade Java - Alura

Projeto desenvolvido durante os estudos de Java, com foco em lógica de programação, estrutura de código e versionamento com Git e GitHub.

## 🚀 Objetivo

Praticar conceitos básicos da linguagem Java, incluindo:

- Variáveis
- Tipos de dados
- Estrutura de saída (System.out.println)
- Organização de código

## 🛠️ Tecnologias utilizadas

- Java
- Maven
- Git
- GitHub
